The goal is the fit \(\sin x\) using a cubic polynomial.

The data will be generated using by sampling the curve \(y = \sin x\). But this relationship between \(x\) and \(y\) will not be known to the modeler.

Let the data \(\{(x_i, y_i)\}_{i=1}^n\) be given.

Our model is \(y = f(x)\) where

\[ f(x) = a + bx + cx^2 + d x^3 \]

, where \(a, b, c, d\) are the free parameters to minimizer over.

The minimization problem is therefore min $_{i=1}^{n} ^2 : a, b, c, d } $

To minimizer, we use gradient descent method. Define \(\omega = (a, b, c, d)\)

\[ L(\omega) = \sum_{i=1}^{n} \abs{y_i - f_\omega(x_i)}^2 \]

and then use the scheme \[ \omega_{n+1} = \omega_n - \eta \nabla_\omega L \]
